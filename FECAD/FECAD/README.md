O projeto está dividido nas seguintes áreas principais:

- **/frontend**: Aplicação web desenvolvida com Next.js.

## Instruções Iniciais

1. Para configurar o frontend, navegue até a pasta `frontend` e instale as dependências (ex.: `npm install` ou `yarn`).
2. Para configurar o backend, navegue até a pasta `backend`, instale as dependências e configure o banco de dados especificado no Prisma.
